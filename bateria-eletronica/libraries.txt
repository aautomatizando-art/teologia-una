# Bibliotecas necessarias
# Instalar via Arduino IDE: Sketch > Incluir Biblioteca > Gerenciar Bibliotecas

# NENHUMA biblioteca de terceiros e necessaria.
# SD, SPI e o driver I2S (driver/i2s.h) ja vem inclusos no ESP32 Arduino Core.
# A saida MIDI e feita "na mao" via Serial2 (bytes MIDI puros), sem lib externa.

# Placa ESP32 (Arduino IDE > Preferencias > URLs adicionais):
# https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json
# Ferramentas > Placa > Gerenciador de Placas > "ESP32 by Espressif Systems" >= 2.0.0

# Placa deste projeto: ESP32-S3-N16R8 (16MB flash, 8MB PSRAM Octal)
# Selecione: "ESP32S3 Dev Module"
# As amostras de audio sao carregadas inteiras na PSRAM -- sem PSRAM
# habilitada corretamente o firmware principal (esp32_bateria_eletronica)
# nao vai conseguir tocar os sons (mas o teste_calibracao_piezo funciona
# em qualquer ESP32/ESP32-S3, com ou sem PSRAM).

# Configuracoes obrigatorias em Ferramentas (Arduino IDE), placa ESP32S3:
#   PSRAM: "OPI PSRAM"   <-- IMPORTANTE: a N16R8 usa PSRAM Octal, nao Quad.
#                             Se deixar em "QSPI PSRAM" ou "Disabled" a
#                             PSRAM nao e detectada.
#   Flash Size: "16MB"
#   Partition Scheme: qualquer esquema de 16MB com espaco de app suficiente
#   USB CDC On Boot: "Enabled" (usa a porta USB nativa para o Serial Monitor)
